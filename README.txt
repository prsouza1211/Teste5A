Informações importantes a respeito do projeto. 
-----------------------------------------------

Na pasta raiz, estão localizados os arquivos de view e também os do Composer (gerenciador de dependências).

Em assets, estão todos os scripts js e css utilizados no projeto. 

Em core, há 3 outras pastas (class que seria o Model, controller e database, onde está a classe de conexão com o BD MySQL). 

Em src, conforme solicitado, o script de criação do BD e uma representação gráfica do modelo de dados. 

Em vendor, estão todas as dependências geradas pelo Composer (Bootstrap, jQuery e jQuery UI).
